# Submission Copy

LaunchLab is a founder-focused AI experiment builder. It turns one rough growth
goal into a guided flow: active goal, ranked test queue, landing-page variant,
result signal, analytics, and a shipped experiment log.

The demo is built as a usable product surface instead of a static portfolio
page. A reviewer can open the app, generate experiments, move through the
builder flow, ship one, see shipped and selected states change, compare A/B
landing copy, inspect mock conversion metrics, and read the case study behind
the product decisions.

The implementation uses Next.js App Router, strict TypeScript, Tailwind CSS v4,
Radix primitives, React Hook Form, Zod, Zustand persistence, Recharts, Biome,
Vitest, and Playwright smoke tests. Everything runs locally with free tooling
and no paid service dependency.
