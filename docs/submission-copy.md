# Submission Copy

LaunchLab is a founder-focused AI experiment cockpit. It turns a rough growth
goal into pipeline lanes of startup experiments, selected insights,
landing-page variants, analytics, and a shipped experiment log.

The demo is built as a usable product surface instead of a static portfolio
page. A reviewer can open the app, generate experiments, ship one, see shipped
and selected states change, compare A/B landing copy, inspect mock conversion
metrics, and read the case study behind the product decisions.

The implementation uses Next.js App Router, strict TypeScript, Tailwind CSS v4,
Radix primitives, React Hook Form, Zod, Zustand persistence, Recharts, Biome,
Vitest, and Playwright smoke tests. Everything runs locally with free tooling
and no paid service dependency.
